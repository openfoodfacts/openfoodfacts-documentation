# Knowledge Panel System

<Callout type="info">
The Knowledge Panel system provides structured information about food products through facets, making it easier for users to understand complex product data.
</Callout>

## What are Knowledge Panels?

Knowledge Panels are structured information displays that present product data in an organized, user-friendly format. They help users understand various aspects of food products through categorized information based on different facets.

## Key Features

<Steps>
<Step title="Facet-Based Organization">
Information is organized by facets such as categories, labels, ingredients, and nutritional aspects.
</Step>
<Step title="Multi-Language Support">
Knowledge panels support multiple languages for international accessibility.
</Step>
<Step title="Country-Specific Data">
Panels can display region-specific information based on country parameters.
</Step>
<Step title="API Integration">
Easy integration with external applications through a RESTful API.
</Step>
</Steps>

## API Usage

The Knowledge Panel API allows you to retrieve structured information about products based on various facets. Here's a quick example:

```json title="Example API Request"
GET https://facets-kp.openfoodfacts.org/knowledge_panel?facet_tag=category&value_tag=yogurts&lang_code=en&country=us
```
